# SOAP-Core v0.7 Representation Geometry Benchmark
日期：2026-06-27
一句话定位：从标量训练日志层进入表示几何层，用隐藏层 effective_rank / collapse_score 验证能否拉开 mode_collapse。

## 1. 为什么 v0.7 转向 representation geometry
v0.6.8/v0.6.9 证明标量训练日志层：divergence 可检测、overfit 加 train_val_gap 后可检测，但 mode_collapse 不可稳定检测。标量层已触顶——mode_collapse 的本质是“loss 平静但表示空间坍缩”，单点统计量（loss/grad/output_variance）看不见。v0.7 进入表示几何层：直接度量隐藏层激活分布的几何（effective_rank、representation_variance、collapse_score），攻 mode_collapse 的盲区。

## 2. 指标定义
- **effective_rank**（entropy effective rank）：对隐藏层激活协方差矩阵的特征值 λ_i，p_i = λ_i/Σλ，effective_rank = exp(-Σ p_i log p_i)。值域 [1, n_features]，越高表示表示空间越丰富，越低越坍缩。
- **representation_variance**：隐藏层激活所有元素的总方差（np.var）。
- **collapse_score**（proxy）：1 / effective_rank。eff_rank 越低 score 越高，表示越坍缩。注意这是 proxy 非严格度量。
- **脚本**：soap/apps/training/representation.py（4 函数：activation_covariance / effective_rank_from_covariance / representation_variance / collapse_score）。
- **隐藏层激活**：_make_mlp 是 Sequential，取最后一个 Linear 前的激活（倒数第二层输出）。

## 4. mode_collapse 是否被拉开
1. **representation 指标层：明显拉开（成功）**。mode_collapse 的 collapse_score=0.998，明显高于 normal（0.65）和 overfit（0.30）；representation_variance=0.028，远低于 normal（0.96）和 overfit（0.22）；effective_rank=1.00 近完全坍缩，低于 overfit（3.29）。这与 normal/overfit 明显分离。**v0.7 验收标准达成**。
2. **SOAP d* 维度：也反映区分**。normal=7 / mc=5 / overfit=1，三者 d* 各不同（representation 信号进入 PCA 改变了有效维度）。
3. **但 SOAP 动力学指纹仍不分**。normal/overfit/mc 三者的 skill≈1、det≈1、next A2 主导——几乎相同。SOAP 的 simplex/recurrence 指纹对这三类仍不可分。
4. **mode_collapse vs divergence：仍清晰**。divergence skill 崩溃（0.232）、det 骤降（0.402）、吸引子分裂（A1/A2），与 mc（skill≈1，det=1.0，A2）截然不同。

## 6. 深层结论
攻 mode_collapse，**直接用 representation 几何指标（collapse_score）判定，比把表示信号喂给 SOAP 动力学 pipeline 更有效**。SOAP 的状态空间方法（PCA d* + simplex + recurrence）对动力学失稳（divergence）敏感，对 representation 失稳（mode_collapse）不敏感——后者的区分信号在几何指标本身，不在曲线动力学指纹。representation 几何指标补上了标量层的盲区，但不是通过 SOAP pipeline 实现的，而是作为独立观测量。

## 7. 边界说明
- **toy representation**：CPU 小 MLP 的隐藏层激活，不等于真实大模型 embedding（后者维度高、结构复杂、collapse 形态多样）。
- **collapse_score = 1/effective_rank** 是 proxy，非严格 collapse 度量。
- **divergence 的 representation 指标爆死**（eff_rank→0，collapse→1e4）是训练发散后激活消失的副产物，非典型 collapse。
- **repr-enhanced CSV** 与 v0.6.8/v0.6.9 CSV 并存，未覆盖。

## 8. 下一步方向（v0.7.1）
- 把 representation 指标（effective_rank / collapse_score / representation_variance）抽象成正式的 **Representation Geometry Adapter**，直接判定 mode_collapse（阈值或规则），而非依赖 SOAP 动力学 pipeline。
- 验证**真实大模型 embedding** 的表示几何（HuggingFace hidden states）是否同样可分。
- **多层 representation**：不只看最后一层隐藏，看每层 effective_rank 的演化曲线。
- v0.7 已确立：**失稳检测应分层**——动力学失稳用 SOAP 状态空间，representation 失稳用几何指标直判。后续产品化按此分层。
