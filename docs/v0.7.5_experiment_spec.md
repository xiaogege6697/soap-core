## 0. 定位与边界（修订 4）

**定位**：**baseline deviation detection** —— 检测表示几何偏离 dev normal 基线的程度。

**不宣称**：
- **不宣称** phase-aware effective_rank 能区分**良性 Neural Collapse**（训练后期自然 ETF 收敛）与**病态 collapse**。两者在 effective_rank 上可能不可区分，本方法只报"偏离基线"，不判良性/病态。
- checkpoint 非独立，**不宣称 conformal 严格 FPR**（SPCI 思想），只借鉴 CADE 多层组合。
- collapse_score = 1/effective_rank 信息重复，**不作第二票**。

## 2. 三方法（修订 1/2）

特征统一 `log(effective_rank)`。`normalized_phase` t = checkpoint_idx/(n_checkpoints−1) ∈ [0,1]。

### 方法 A：v0.7.4 static per-layer threshold（baseline）
- 每 layer 用 dev normal 的 `collapse_score` 99 分位作静态阈值；run-level any_layer。
- 对照用，看 phase-aware 是否真降误报。

### 方法 B：phase-knots baseline（核心，修订 1/2）
- **8 phase knots**：t = 0, 1/7, 2/7, ..., 1（均匀 8 点）。每 `(layer, knot)` 用 dev normal（42-53）的 `log(eff_rank)` 算 **median 与 MAD_scaled（1.4826·MAD）**。
- **knot 间线性插值**：任意 phase t 的 median/MAD 由相邻两 knot 线性插值（t<0 用 knot0，t>1 用 knot7）。
- **MAD layer floor（修订 2）**：每 layer 的 MAD 取 `max(MAD_interpolated, floor_layer)`，floor_layer = 该层 dev normal `log(eff_rank)` 全体 raw MAD_scaled 的中位（防个别 knot MAD 过小致 anomaly 爆炸）。
- anomaly score：`a(l,ckpt) = |log(eff_rank) − median_interp(l,t)| / (MAD_floored(l,t) + eps)`。
- **run-level 聚合**：`run_score = max over (l,ckpt) of a`（checkpoint 非独立）。
- run-level threshold：dev normal run_score 的 run_q 分位。

### 方法 C：B + persistent（修订 1）
- 在 B 的 run_score 基础上，额外要求 persistent：某 layer 连续 ≥3 checkpoint 的 `a(l,ckpt)` 超 run_threshold。
- run 检出 = B 超 threshold 且 persistent 命中。

## 3. 评估（修订 3/6，无泄漏）

### held-out normal FPR（54-73，20 run）
- A/B/C 各算误报数（0-20）+ **Wilson 95% CI**。
- **exact McNemar paired comparison（修订 6）**：A/B/C 两两在**相同 20 held-out runs**上的检出（0/1）配对，exact McNemar（binomial，n_discordant 小用二项精确而非 χ²）。
- run_q 主结果 = **0.95**；**q=0.90 仅 sensitivity**；**不用 q=0.99**（修订 3）。

### held-out induced 检出（45-47，3 run）
- A/B/C 各算检出数（0-3），**不为保 3/3 反向调参**。
- 诊断特征（非硬规则）：首现层、发生 phase、持续时间、传播顺序。

### 成功标准
- **不是 FPR 归零**，是**无泄漏的误报—检出权衡**（A/B/C 在 held-out 的 trade-off + McNemar 显著性）。

## 4. 边界
- 区分 **sample-pooled effective_rank**（本实验）与 **token-rank collapse**（Transformer 文献，NeurIPS 2024）。
- `layer_4→5→6` 传播仅诊断特征，**不作硬规则**。
- **class-conditional geometry 延后 v0.7.6**（修订 7），不改当前实验字段。
- 证据边界：限于 controlled rank-1 benchmark（预训练 DistilBERT 合成任务）；自然 mode collapse 未验证；FPR 小样本不外推。
