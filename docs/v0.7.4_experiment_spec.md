# v0.7.4 实验 Spec — task34 `hf_repr_experiment.py`

> 本文件是**实现契约**，主线程撰写（设计决策不可外包）。MiMo 严格据此实现。
> 阶段：spec → MiMo 实现 → 主线程审核 + 单 seed pilot → 冻结干预强度 → 正式 3 seed。

## 0. 目标

对 `distilbert-base-uncased` 微调，在**固定 probe set** 上**逐层**记录 representation metrics
（`effective_rank` / `representation_variance` / `collapse_score`），对比 **normal** vs
**controlled induced collapse** 两种 condition，产出 per-seed × per-condition × per-layer ×
per-checkpoint 的 CSV，供 task35 做 leave-one-seed-out 校准 + FPR + 传播路径分析。

**本脚本只产出 metrics CSV，不含任何 detector 阈值逻辑**（阈值归 task35）。

## 1. 复用约定（禁重定义，确保与 v0.7.3 对齐）

```python
from soap.apps.training.representation import (
    activation_covariance, effective_rank_from_covariance,
    representation_variance, collapse_score,
)
from soap.apps.training.torch_runs import _safe_float
```

- 三指标定义与 `representation.py` 完全一致；collapse_score = 1/effective_rank。
- `_safe_float` 防 NaN：传入 (新值, 上次有效值)，新值非有限时回退上次值。
- 训练用 mini-batch（normal 也用 batch，避免 v0.6.8 的 full-batch 无噪声问题）。

## 2. 数据 `make_structured_dataset()`

- **结构化英文模板，5 类**（禁用随机 token ID — 脱离真实语义分布）。
- 每类 1 个句式骨架 + 词槽词表，**类间词汇不重叠**、类内语法一致、类间结构可分。
- 骨架示例（词表由实现填充，组合数 ≥ 350/类以撑起 320 条不退化）：
  - class0 travel : `the traveler visited {place} {freq} during the {season} holiday`
  - class1 food   : `the chef served {adj} {dish} to the guests at dinner`
  - class2 weather: `the forecast predicted {adj_w} weather across the {region}`
  - class3 sports : `the athlete won the {sport} event at the {venue}`
  - class4 tech   : `the engineer tested the {device} with the new software`
- **数据生成 seed 固定 = 0**（`DATA_SEED=0`），所有训练 seed 共用同一数据集 ——
  只改模型初始化 / dropout / 训练 batch 顺序，不改数据本身。
- 规模：`train=1600`（320/类），`val=400`（80/类），`probe=128`（固定，~25/类，独立于 train/val）。
- 输出 `(texts_train, labels_train, texts_val, labels_val, texts_probe, labels_probe)`，全部 deterministic 可复现。

## 3. tokenizer

```python
from transformers import DistilBertTokenizer
tok = DistilBertTokenizer.from_pretrained("distilbert-base-uncased")  # 走缓存
# encode: padding="max_length", truncation=True, max_length=32 → input_ids + attention_mask
```

## 4. 模型 `DistilBertClassifier(nn.Module)`

```python
backbone = DistilBertModel.from_pretrained("distilbert-base-uncased", output_hidden_states=True)
classifier = nn.Sequential(nn.Dropout(p), nn.Linear(768, 5))
```

- `forward(input_ids, attention_mask) -> (logits, hidden_states_tuple)`
  - `hidden_states` = backbone 输出 7 个张量：`[embedding, layer1..layer6]`，每个 `(B, 32, 768)`
  - pooled = **mask 加权 meanPool**（忽略 pad token）over last_hidden → `(B, 768)`
  - `logits = classifier(pooled)`
- **冻结策略（normal 与 induced 完全相同，作为受控变量）**：
  - freeze `backbone.embeddings`
  - freeze `backbone.transformer.layer[0]`、`layer[1]`、`layer[2]`
  - 可训练：`layer[3]`、`layer[4]`、`layer[5]` + classifier
  - 注意冻结后前向仍产出全部 hidden_states（用于逐层 probe）。

## 5. probe set 逐层 metrics（每 checkpoint，eval 模式）

- 固定 probe set 128 条 forward，`output_hidden_states=True`
- 对每层 `hidden_states[i]`（i=0..6，共 7 层，命名 `layer_0`..`layer_6`，layer_0=embedding）：
  - mask 加权 meanPool over seq_len → `(128, 768)`
  - `cov = activation_covariance(pooled_128x768)`
  - `effective_rank = _safe_float(effective_rank_from_covariance(cov), last)`
  - `collapse = _safe_float(collapse_score(pooled_128x768), last)`  # 内部调 eff_rank，用 pooled
  - `rep_var = _safe_float(representation_variance(pooled_128x768), last)`
- **三个指标统一在 pooled `(128,768)` 矩阵上算**，自洽。
- 记录每层（layer_0..layer_6）一行。

## 6. variance contraction loss（仅 induced，这是 collapse 干预机制）

- 施加点：**transformer block 3 的输出 = `hidden_states[4]`**（第一个可训练层输出，中层）
- token-level per-feature variance：
  ```
  H = hidden_states[4]                    # (B, 32, 768)
  H_flat = 按 attention_mask 展平非 pad token → (N_tok, 768)
  var_per_feat = var(H_flat, dim=0)       # (768,)
  L_contract = mean( -log(var_per_feat + 1e-6) )   # 鼓励方差趋小 → collapse
  ```
- **总 loss**：`L_total = L_cls + λ * L_contract`，`L_cls = CrossEntropy(logits, labels)` 保留。
- `λ`（lambda_contract）**pilot 确定**，CLI 可调，默认起始候选 `0.1`。
- normal condition：`L_total = L_cls`（无 contraction，λ 不生效）。
- weight_decay：normal 与 induced **都 = 0**（collapse 纯靠 contraction loss，隔离变量，不靠 wd 收缩参数）。

## 7. 训练循环

- seed = {42, 43, 44}（model init + dropout + batch shuffle）；数据用 DATA_SEED=0 固定。
  - 每 seed 开头 `torch.manual_seed/np.random.seed/random.seed` 全设。
- `steps=120`，`batch_size=16`，`record_every=3` → **40 checkpoints**。
- optimizer = `AdamW(可训练参数, lr=2e-5, weight_decay=0.0)`。
- 每 step：`model.train()` → 取 shuffled batch(16) → `logits, hs = model(...)` →
  `L_cls = CE` → induced 加 `λ*L_contract(hs[4], mask)` → backward → `grad_norm` → step → zero_grad。
- 每 `record_every` steps（eval 模式）：
  - probe set 算逐层 metrics（第 5 节）
  - `val_loss` 在 val set 上算 CE
  - 对每层写一行 CSV（layer_0..layer_6），timestamp=`step_{step}`

## 8. CSV schema

```
HEADER = ["seed","condition","timestamp","layer_name","loss","val_loss",
          "grad_norm","learning_rate","effective_rank","representation_variance","collapse_score"]
```
- `seed` = 训练 seed；`condition` = `normal`/`induced`
- 文件名：`examples/hf_repr_seed{seed}_{condition}_multilayer.csv`
- 每 checkpoint × 每层一行（40 × 7 = 280 行/文件）。

## 9. CLI

```
python -m soap.apps.training.hf_repr_experiment \
    --seed 42 --condition {normal|induced} \
    --steps 120 --batch-size 16 --record-every 3 \
    --probe-size 128 --device auto \
    --lambda-contract 0.1 \
    --output examples/hf_repr_seed42_induced_multilayer.csv
```
- `--device auto`：`mps` 可用→`mps`；`cuda` 可用→`cuda`；否则 `cpu`。
- 启动时**打印实际 device 到 stdout**（记录用，不宣称跨设备逐位可复现）。

## 10. 数值安全

- 所有写入 CSV 的浮点经 `_safe_float` 兜底。
- contraction loss 的 `eps=1e-6` 防 `log(0)`。
- 出现 NaN/Inf 时 `_safe_float` 回退上次有效值，不中断（与 v0.7.3 一致）。
