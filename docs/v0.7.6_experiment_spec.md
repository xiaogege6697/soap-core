## 0. 定位

v0.7.5 的 phase-aware B 回答"是否偏离 normal baseline"，**无法区分**：
- **良性类内聚焦**（Neural Collapse 趋近）：类内方差收缩，但类中心保持多维分离、ETF geometry、accuracy。

v0.7.6 引入 class-conditional geometry，尝试区分二者。

## 3. 核心指标（公式冻结）

设 probe set 表示 X（N×D，N=128，D=768），labels y（5 类），每 class c 的样本 X_c，均值 μ_c，全局均值 μ；K=5。

### 3.1 NC1 ratio（类内收缩，良性 → 0）
```
within_class_variance = Σ_c Σ_{x∈X_c} ||x − μ_c||² / N
total_variance        = Σ_x ||x − μ||² / N
NC1_ratio = within_class_variance / total_variance
```
良性 NC：NC1 → 0（类内坍缩到类中心）。

### 3.2 class-centroid effective rank（类中心分离，良性 → K）
```
M = [μ_1, ..., μ_K]^T  (K × D)，去全局均值中心化
centroid_eff_rank = effective_rank_from_covariance(cov(M))   # 复用 representation.py
```
良性 NC：centroid_eff_rank → K（5 类中心多维分离）。病态：→ <K。

### 3.3 Fisher separation（良性 → 大）
```
S_b = Σ_c n_c (μ_c − μ)(μ_c − μ)^T   # between-class scatter
S_w = Σ_c Σ_{x∈X_c} (x − μ_c)(x − μ_c)^T  # within-class scatter
Fisher = trace(S_b) / (trace(S_w) + eps)
```

### 3.4 ETF deviation（类中心 simplex 偏离，良性 → 0）
```
M_n = M 行归一化（每行 unit norm）
G = M_n @ M_n^T  (K×K 内积矩阵)
G_etf: 对角 1，非对角 −1/(K−1)   # ETF 理想内积
ETF_deviation = ||G − G_etf||_F / ||G_etf||_F
```
良性 NC：类中心 → ETF simplex，deviation → 0。

### 3.5 validation accuracy / loss（任务功能，良性保持）
probe set 分类正确率 + cross-entropy（沿用训练 loop 的 val）。

## 4. 预期区分（不强制成功，负面结果保留）

| 指标 | 良性类内聚焦 | 病态全局坍缩 |
|---|---|---|
| NC1 ratio | ↓（类内收缩） | ↓ 或不变 |
| centroid eff rank | 保持（~K） | ↓（<K） |
| Fisher | 保持/↑ | ↓ |
| ETF deviation | ↓（趋 simplex） | 不变/↑ |
| accuracy | 保持 | ↓ |

**判定**：若 induced（controlled rank-1）相对 normal 在 centroid rank / Fisher / accuracy 同步恶化 → 病态；若只有 NC1 降而 centroid/ETF/accuracy 保持 → 良性聚焦形态。
