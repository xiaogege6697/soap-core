# SOAP-Core v0.7.6 class-conditional geometry benchmark 科学文档

> 状态：骨架版（边界/公式/设置/预期/复现已定稿；结果与判定段等 task15 分析后填充）。
> v0.8 继续冻结；class-conditional 是否进默认待评估。

## 摘要
v0.7.5 phase-aware baseline deviation detection（方法 B）在 controlled rank-1 benchmark 上实现 held-out normal FA 0/20、induced Det 3/3（McNemar p=0.000002 vs 静态 A），但**仅检测 baseline 偏离**，按其术语边界**不宣称**能区分"良性类内聚焦（Neural Collapse 趋近）"与"病态全局坍缩"。v0.7.6 引入 class-conditional geometry（NC1 ratio / centroid effective rank / Fisher separation / ETF deviation / validation accuracy），尝试在 controlled benchmark 内补一层语义证据：若 induced 仅 NC1 降而 centroid rank/Fisher/ETF/accuracy 保持 → 良性聚焦形态；若同步恶化 → 病态 collapse。

## 证据边界（承接 v0.7.5，严格不外推）
- **induced collapse 性质**：controlled rank-1 intervention（alpha 0→0.98 前向投影），**不代表**自然 mode collapse 已被验证。
- **实验对象**：预训练 DistilBERT 在 synthetic 5 类分类任务上的微调，结论严格限定于该模型/任务/规模，**不可外推**。
- **小样本**：dev normal 12 / held normal 20 / induced 6 runs，统计量（mean/std/百分比）**不可外推**为总体。
- **分离来源**：rank-1 intervention 在数学空间上天然压低 centroid rank / 抬高 NC1 收缩，与 class-conditional 指标部分对齐；任何 observed 分离**不能直接归因**于指标对自然 collapse 的判别力。
- **不要求成功区分**良性/病态；负面结果保留（与 v0.7.4/5 一致）。
- **术语**：本文称 "class-conditional geometry 观察"，**不宣称** class-conditional 指标已在自然 collapse 上验证良性/病态区分。

## 实验设置
- **模型/软件/加载**：同 v0.7.5（distilbert-base-uncased revision `12040accade4e8a0f71eabdb258fecc2e7e948be`，transformers 5.12.1，torch 2.12.1，`local_files_only=True`，冻结 embeddings + 前 3 层）。
- **数据**：5 类结构化英文模板，train 1600 / val 400 / probe 128，`data_seed=0` 固定。
- **v0.7.6 三池修正**：train / validation / probe 严格分离；probe **不入**训练索引（无泄漏）；validation_accuracy 基于**独立 val split**（非 probe/train），采用 normal 前向（alpha=0，即去掉 rank-1 干预的推理路径）。
- **种子与条件**：normal 42-73，induced(condition_b) 42-47；alpha schedule 同 v0.7.4/5。
- **划分**：dev normal 42-53（建参考）/ held normal 54-73（评估）/ dev induced 42-44 / held induced 45-47。
- **记录**：每 3 步记录（40 checkpoint）× 7 层（embedding + 6 transformer），class-conditional 指标基于 probe set（128，~25/类）。每 run CSV 280 行 = 40 ckpt × 7 层。

## 五项 class-conditional 指标（公式冻结，详见 `docs/v0.7.6_experiment_spec.md`）
设 probe set 表示 X（N×D，N=128，D=768），labels y（5 类），类 c 样本 X_c，类均值 μ_c，全局均值 μ，K=5。

1. **NC1 ratio** = `Σ_c Σ_{x∈X_c}||x−μ_c||²/N` ÷ `Σ_x||x−μ||²/N`（类内收缩，良性→0）
2. **centroid effective rank** = `effective_rank_from_covariance(cov(M−μ))`，M=K×D 类中心矩阵（中心化后秩≤K−1=4；良性参考 K−1 非 K）
3. **Fisher separation** = `trace(S_b)/(trace(S_w)+eps)`，S_b=Σn_c(μ_c−μ)(μ_c−μ)^T（良性→大）
4. **ETF deviation** = `||G−G_etf||_F/(||G_etf||_F+eps)`，G=M_n@M_n^T（行归一化），G_etf 对角1非对角 −1/(K−1)=−0.25（良性→0）
5. **validation accuracy** = 独立 val split 分类正确率，normal 前向（任务功能，良性保持）

## 预期区分（不强制成功，负面结果保留）
| 指标 | 良性类内聚焦 | 病态全局坍缩 |
|---|---|---|
| NC1 ratio | ↓（类内收缩） | ↓ 或不变 |
| centroid eff rank | 保持（~K−1） | ↓（<K−1） |
| Fisher | 保持 / ↑ | ↓ |
| ETF deviation | ↓（趋 simplex） | 不变 / ↑ |
| validation accuracy | 保持 | ↓ |

**判定原则**：induced（controlled rank-1）相对 dev_normal 参考，若 centroid rank / Fisher / accuracy 同步恶化 → 病态；若仅 NC1 降而 centroid/ETF/accuracy 保持 → 良性聚焦形态。阈值/参考**只用 dev normal(42-53)**；held-out 仅评估。

## 结果
induced（dev+held condition_b 合并 6 seeds）vs dev_normal(12) 末段（最后 5 checkpoint）逐层对比（完整见 `examples/_classcond_analysis.txt`）。

**held_normal 落在 dev_normal mean±std 内**（baseline 稳定）；held_induced 与 dev_induced 一致（可重复）。

**冻结层 layer_0-3**（权重不变）：induced 与 normal **全部持平（±0%）**，自洽。

**可训练层 layer_4-6**（rank-1 干预注入处）：

| 层 | NC1 | centroid eff rank | Fisher | ETF | val_acc(normal前向) |
|---|---|---|---|---|---|
| layer_4 | 0.170→0.013 (↓92%) | 3.89→**1.11** (↓71%) | 4.90→85.9 (↑1655%) | 1.99→2.16 (↑9%) | 1.0→1.0 |
| layer_5 | 0.065→0.044 (↓31%) | 3.88→3.14 (↓19%) | 14.6→21.7 (↑49%) | 1.28→1.74 (↑35%) | 1.0→1.0 |
| layer_6 | 0.036→0.021 (↓43%) | 3.99→3.87 (↓3%) | 26.9→48.2 (↑80%) | 0.39→0.51 (↑31%) | 1.0→1.0 |

## 判定
induced（controlled rank-1）在可训练层（尤 layer_4）表现为**几何坍缩**：NC1↓（类内收缩）+ centroid eff rank↓（layer_4 →1.1，几乎压到 1 维，即 rank-1 投影的直接几何证据）+ ETF↑（偏离 simplex，非良性收敛）。但同时：

- **val_accuracy 持平 1.0**（normal 前向 alpha=0）：仅说明当前合成分类任务的 validation accuracy 保持 1.0。**不能由此推断**模型通用功能、自然鲁棒性或权重功能保持——一维表示仍可能完成这个简单五分类任务。collapse 的「前向注入」性质不等于「权重学到」。
- **Fisher 暴涨**（layer_4 ↑1655%）：rank-1 投影把 within-class scatter 压到极低，致 trace(S_b)/trace(S_w) 失真——此为**数学假象**，**不可解读**为分离度改善。

**综合**：induced 既非典型**良性 NC**（无 ETF 收敛、无 centroid rank 保持），也非典型**病态 collapse**（accuracy 未恶化）。它是 controlled rank-1 intervention 注入的几何坍缩；当前合成五分类 val_acc 仍 1.0，但一维表示即可完成该简单任务，不可据此推断权重/通用功能保持。

**结论：在当前 controlled benchmark 中，class-conditional geometry 不能判定良性/病态自然 collapse。** rank-1 intervention 与 effective-rank / centroid rank 在数学空间对齐（投影天然压维），observed 几何坍缩不能归因为指标对自然 collapse 的判别力；Fisher 暴涨是投影假象。此结论与 v0.7.5 边界一致。

## 尚未验证（承接 v0.7.5）
- 跨任务 / 跨模型 / 自然 mode collapse 条件下有效性均未验证。
- class-conditional 参考（dev_normal 末段 mean±std）须按特定模型/层/任务重校准，**不可直接迁移**。
- validation_accuracy 用 normal 前向（去掉干预）；当前合成五分类上 val_acc=1.0 **不可推断**模型通用功能/自然鲁棒性/权重功能保持（一维表示仍可完成简单任务）。condition_b 的 rank-1 干预为前向注入，去掉干预后表示能否支撑复杂任务未验证，不可仅凭该 accuracy 下结论。
